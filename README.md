# Shopping Cart Application

**Created by Triangle Dev Team**

---

## Description
This is a **Shopping Cart application** built with **React** for the frontend and **Node.js/Express** for the backend.  
The app allows users to browse products, add them to a shopping cart, and manage their orders.  

---

## Features
- Browse and view products  
- Add/remove items from the shopping cart  
- Update item quantities  
- Responsive and user-friendly interface  
- Backend API built with Express for product and cart management  

---

## Technologies Used
- **Frontend:** React, CSS  
- **Backend:** Node.js, Express  
- **Other:** dotenv, CORS
